# 方法论：五看六定（Five-Look Six-Set）

> **雨哥（雨哥聊海外销售）改造版** · 海外销售方法论金字塔「第 5 层 · 市场经营」

## 一、起源

"五看六定"是华为体系内成熟的**市场战略分析框架**，最初用于公司层面的业务规划（BLM · Business Leadership Model）的输入环节。后被广泛应用于：

- 品牌出海（中国品牌进入海外市场前的尽调）
- 国际市场进入（New Market Entry）
- 海外销售战略制定（Annual Sales Planning）
- 代理商体系搭建（Channel Strategy）
- 区域生意复盘（Quarterly / Annual Business Review）

**核心思想**：先看清市场（Look），再定打法（Set）。**没有市场洞察的策略是空中楼阁；有了洞察才能定具体打法。**

---

## 一·5 雨哥改造版 —— 从"业务规划工具"降维到"海外销售战术工具"

华为原版的五看六定是给**业务部门做年度规划**用的——输入是公司战略，产出是 SBU 业务计划，输出周期是季度 / 年度。

**雨哥（雨哥聊海外销售）的改造版** 把它降维到了"个人销售 / 求职者层面"—— 30 分钟到 90 分钟内，给个人销售或求职者一份能立刻拿去用的弹药。

### 华为原版 vs 雨哥改造版

| 维度 | 华为原版 | 雨哥改造版 |
|---|---|---|
| 使用者 | 业务部门负责人 / SBU 老大 | 一线销售 / 求职者 / GTM 个人 |
| 输入 | 战略意图 + 公司资源盘点 | 一家公司 + 调研目的（拜访 / 尽调 / 面试 / 决策）|
| 产出周期 | 季度 / 年度 | **30 分钟到 90 分钟** |
| 数据源 | 内部市场调研 + 咨询报告 | **中英文双搜 + 公开行业报告 + 官网爬** |
| 竞品分析 | 战略对手 | **强制 ≥ 5 家**（看清结构）|
| 信息可靠性 | 由分析师标注 | **强制区分事实 / 推测**（用 AI 时尤为关键）|
| 输出最后一步 | 战略意图 → BLM 输入 | **3 个核心洞察 + 3 个关键问题**（行动弹药）|
| 典型场景 | SBU 业务规划 / 年度战略 | 客户拜访 / 代理商尽调 / 求职面试 / 内部 review |

### 雨哥改造的 4 个关键设计

1. **强制中英文双搜** —— 中文搜公司背景 / 融资 / 创始团队，英文搜海外市场数据 / 竞品 / 客户案例。**单一语言会漏数据** —— 这是 13 年海外销售实战发现的最大坑。
2. **竞品 ≥ 5 家硬性门槛** —— 华为有内部数据库，可以一站拿到竞争格局。个人版没这个条件，**5 家是看清结构的最低门槛** —— 少于 5 家就是"以偏概全"。
3. **3 个洞察 + 3 个问题作行动弹药** —— 华为版输出战略意图，雨哥版输出**能立刻说出口的话**：客户拜访开场 / 谈判切入 / 面试发言 / 反问考察对方，都用得上。
4. **避免沉没成本陷阱** —— 个人在做"是否进入这个市场 / 是否签这家代理 / 是否选这家公司"决策时，容易陷入"已经投入了所以继续"。雨哥版强制 **先看 hard constraint**（资质 / 认证 / 渠道壁垒 / 利益冲突），**再谈匹配度梯度**。

### 改造背景

雨哥在普渡机器人（亚太区年营收破亿期间）和早期海外销售实战中，大量使用过华为版五看六定。作为 **一人 / 小团队海外销售或求职者** 使用时，发现 3 个 gap：

- 华为版默认有"团队 + 数据库 + 内部资源"，个人版没有 → **需要轻量化的数据采集 SOP**（中英文双搜 + 公开报告）
- 华为版输出是"战略文档"，个人版需要的是"明天就能说出口的话" → **需要把战略洞察翻译成可执行话术弹药**
- 华为版竞品分析不强制数量，个人版容易"列 2-3 家就收工" → **需要硬性门槛防止偷懒**

这 3 个 gap 就是雨哥改造版要解决的问题，也是这个 Skill 的设计源头。

## 二、五看（市场洞察 · 5 个维度）

| 看什么 | 核心问题 | 输出 |
|---|---|---|
| **看行业趋势** | 这个行业未来 3-5 年会怎么变？什么在驱动？ | 趋势清单 + 政策/技术/社会驱动 |
| **看市场规模** | 目标区域市场多大？增速多少？细分结构？ | TAM / SAM / SOM 三层数据 + CAGR |
| **看竞争格局** | 谁在打？市占率如何？我能在哪里切入？ | 至少 5 家竞品对比表（产品 / 价格 / 渠道 / 客户）|
| **看客户需求** | 客户是谁？怎么买？看重什么？ | 客户分层 + 决策链 + 购买逻辑 + 关键 KPI |
| **看自身** | 我的优势 / 短板 / 机会 / 威胁是什么？ | SWOT 矩阵 + 真实差距评估 |

### 五看的常见误区

❌ **只看"看行业"和"看市场"，忽略"看客户" / "看竞争" / "看自身"** —— 调研做成"行业研究报告"而不是"我能怎么打"
❌ **"看自身"写得太自夸** —— SWOT 里 W（弱点）和 T（威胁）必须真实，否则六定全错
❌ **竞品只列 2-3 家** —— 看不出结构，看不出空白机会
❌ **"看客户"停留在描述，不画决策链** —— 决策链才是销售真正的作战地图

## 三、六定（业务策略 · 6 个落地动作）

业务侧又叫 **4P + 1S + 1C**：

| 定什么 | 经典理论 | 内容 |
|---|---|---|
| **定目标市场** | STP - Targeting | P1 / P2 / P3 国家或区域优先级 + 排序理由 |
| **定客户画像** | STP - Segmentation | 客户类型 + 决策人头衔 + 采购逻辑 + 触达入口 |
| **定产品策略**（Product）| 4P - Product | 引流款 / 主推款 / 旗舰款 三层产品组合 |
| **定价格策略**（Price）| 4P - Price | 对标竞品价 + 渠道利润空间 + 折扣节奏 |
| **定渠道策略**（Place）| 4P - Place | 直销 / 代理 / 线上 / 展会 / 行业协会 |
| **定推广策略**（Promotion）| 4P - Promotion | 线上（LinkedIn / 行业论坛）+ 线下（展会 / 协会）+ 内容（KOL / PR）|

> **Service** 和 **Customer**（1S + 1C）作为隐性维度贯穿在六个"定"里 —— 服务承诺影响产品/价格/渠道；客户洞察是所有"定"的基础。

### 六定的常见误区

❌ **跳过"看"直接"定"** —— 闭门造车，定的策略落不了地
❌ **"定价"只对标一家竞品** —— 价格带要至少有 3 个锚点（高端 / 中端 / 入门）
❌ **"定渠道"只想线上** —— 海外 B2B 大单基本走线下展会 + 经销商 + 行业协会，纯线上是消费品逻辑
❌ **"定推广"等同于"投广告"** —— 内容（PR / 行业评测 / 客户案例）和关系（行业协会 / KOL）权重远大于付费投放

## 四、五看六定 vs 其他经典框架

| 框架 | 适用场景 | 跟五看六定的关系 |
|---|---|---|
| **STP** | 营销定位（细分 / 目标 / 定位） | 五看六定的"看客户 + 定目标市场 + 定客户画像"是 STP 的具体化 |
| **4P / 4P+** | 营销组合 | 六定的"定产品 / 价格 / 渠道 / 推广"就是 4P |
| **SWOT** | 自身评估 | 五看的"看自身"输出 |
| **PESTEL** | 宏观环境 | 五看的"看行业趋势"补充 |
| **波特五力** | 行业竞争结构 | 五看的"看竞争格局"深化 |
| **BLM（业务领先模型）** | 完整业务规划 | 五看六定是 BLM 的"市场洞察 + 战略意图"输入环节 |

**心智结构**：五看六定是"骨架"，其他经典框架可以作为"血肉"嵌入特定章节。

## 五、应用案例

### 案例 1：海外销售面试准备

**场景**：候选人面试某家中国品牌的欧洲销售岗位

**用法**：
- **五看**：用 30 分钟产出"行业 + 市场 + 竞争 + 客户 + 公司自身" 5 维分析
- **六定**：基于五看输出"如果我入职第一年怎么打"具体策略
- **3 个洞察 + 3 个好问题**：作为面试现场可直接说出口的弹药

**效果**：候选人对面试官说"我看到贵司在欧洲已经覆盖 UK / France / Germany，但意大利 / 西班牙 / 北欧仍是空白——这是我能带来 0→1 增量的地方"，面试官当场被打动。

### 案例 2：新市场进入决策

**场景**：某 SaaS 品牌评估是否进入东南亚

**用法**：
- **五看**：先调研东南亚 6 国（新马泰越印菲）的市场规模 + 竞品 + 客户
- **六定**：基于五看排序 P1（首选国家） / P2（次选） / P3（暂缓），并明确"用什么产品 + 什么价格 + 什么渠道 + 什么推广"

**效果**：避免拍脑袋"all-in 东南亚"，转为"先攻新加坡 + 印尼 P1，泰国 P2，马来菲律宾 P3 暂缓"。

### 案例 3：代理商谈判前尽调

**场景**：跟某海外代理签约前，做对方的尽调

**用法**：
- **五看**：调研代理商所在国家市场 + 代理已代理的品牌 + 决策链
- **六定**：基于尽调结果定"是否签独家 + 价格策略 + 推广支持"

**效果**：避免"急着签了再说"，提前发现代理商已代理 2 家直接竞品（利益冲突），调整为非独家分销。
