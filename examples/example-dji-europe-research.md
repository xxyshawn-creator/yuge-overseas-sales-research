## 一、五看（市场洞察）

### 看一：行业趋势

欧洲消费级无人机行业 2024-2026 同时发生 4 个结构性变化：

- **EASA 统一监管落地（2024 起）**：欧盟 27 国 + EFTA 国家统一三级（Open / Specific / Certified）+ 七级 C-class（C0-C6）。**核心变量**：
  - C0 级（≤250g · Mini 系列）→ **无需操作员注册 + 无需理论考试**，监管最友好
  - C1（≤900g · Air 系列）→ 需操作员注册 + 理论考试
  - C2（≤4kg · Mavic / Inspire）→ 距离人群至少 30m
  - **C 标识转换期 2024 年结束**：旧机型继续可飞但限制多，**新机型 C 标识销售优势显著**
- **微型化加速 + 轻量化卡位**：DJI Mini 系列（≤249g）在欧洲是最大单品 SKU——监管友好 + 价格友好（€349-1,099）+ 学习曲线低
- **AI 自动飞行 + 跟拍成新刚需**：ActiveTrack / FocusTrack / MasterShots 是消费级核心功能 → 决定 DJI vs Autel 的真正差距
- **数据合规收紧**：欧盟对中国硬件的"数据流出"敏感度升高 · DJI 已在欧推出"本地数据模式 + LDM"应对

**宏观驱动**：
- 欧洲 27 国 + 英国 + 瑞士 + 挪威 = 约 5.4 亿人口 / 高 GDP
- 影像创作者经济（YouTube / Instagram / TikTok）拉动消费级
- 房地产 / 婚庆 / 工业巡检 / 农业 4 大 B2B 场景持续增长
- 应急救援 / 警务 / 国防 = Specific / Certified 类目长期增量

**关键判断**：欧洲消费级无人机市场进入"**轻量化 + 监管驱动 + AI 体验竞争**"成熟期，**DJI 仍然主导但需要应对中欧贸易紧张 + 政治敏感性**。

## 三、面试时可讲的 3 个洞察

### 洞察 1：欧洲监管 Open category C0 级（≤250g）是 DJI Mini 系列的天然护城河

"EASA 自 2024 起统一 Open / Specific / Certified 三级监管。**C0 级（≤249g）无需操作员注册 + 无需理论考试 + 大多数公共场所可飞**——这直接让 DJI Mini 系列成为欧洲所有新手 + 礼物买家的几乎唯一合理选择。Autel Nano+ 想抢这个细分但渠道厚度不够。**销售岗在欧洲推产品矩阵时，Mini 系列的'监管 + 价格'双重门槛优势是 DJI 不会被颠覆的真正护城河**。"

### 洞察 2：DJI 在欧洲消费级 60-70% 份额下，**真正的威胁不是产品而是地缘政治**

"Autel 是 DJI 唯一同价位段对手，但渠道厚度只有 DJI 的 1/3-1/4。Skydio / Parrot / Yuneec 已经全部退出消费级。所以在欧洲，**DJI 的'产品 vs 产品'战已经赢了**。真正的不确定性来自三件事：① 中欧贸易紧张可能引发关税升级 ② EU 数据合规对'本地数据模式'的要求收紧 ③ 美国 NDAA 关联效应可能传导欧洲。**销售岗的真正价值不是抢产品份额，而是帮 DJI 在不确定性下守住渠道关系 + 拓展低风险增量市场（农业 / 户外 / 教育）**。"

### 洞察 3：农业 Agras + 工业 Matrice 是 DJI 在欧洲未来 3-5 年的真正增长曲线
